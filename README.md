### 기능
- [x] 첫번째 섹션의 3개의 프로필 카드는 페이지를 열때마다 순서가 임의로 배치되도록 구현하세요.
 
- [x] 두번째 섹션의 풀블리드 배경 이미지는 API를 이용해 임의의 이미지 삽입,
같은 브라우저로 페이지를 다시 열거나 새로고침 해도 기존 이미지가 변경되지 않도록 구현
  
- [x] 두번째 섹션 인풋 valid/invalid 케이스의 유효성 검증

- [x] 세번째 섹션의 필터 기능

- [x] 반응형
